# Flutter Taxi App

โปรแกรมคำนวณค่าโดยสารแท็กซี่ตามอัตราค่าโดยสารประเทศไทย

## การคำนวณ
- ค่าเริ่มต้น 35 บาท
- คิดตามระยะทางแบบขั้นบันได
- ค่ารถติด 3 บาท/นาที
